# Financial-KPI-Extractor

[Live App](https://financial-kpi-extractor.streamlit.app/)

Tech Stack: Python, Streamlit, Groq API, Plotly, LangChain, Llama 3

- Built an automated pipeline to extract unstructured financial data (Revenue/EPS) from news articles using Llama 3.3 (via Groq) and LangChain.
- Engineered a prompt-based extraction logic to parse complex text into structured JSON with 95%+ accuracy.
- Developed a Streamlit frontend with interactive Plotly visualizations for real-time "Actual vs. Expected" analysis.
- It solves the manual data entry problem for financial analysts.
